# Vehicle Search Dashboard

A responsive web application built with **React**, **TypeScript**, and **Vite** that allows users to search and filter a vehicle database. This project features dynamic cascading filters and a clean data table display.

## Features

* **Dynamic Filtering**: Filter vehicles by Country, Type, Make, Model, Year, Engine Size, and Fuel Type. Dropdown options automatically update based on previous selections.
* **Interactive Results**: Displays search results in a clean table using `@tanstack/react-table`.
* **Responsive Design**: Optimized for different screen sizes.
* **Fast Performance**: Powered by Vite for quick development and building.

## Tech Stack

* **Framework**: [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
* **Build Tool**: [Vite](https://vitejs.dev/)
* **Styling**: CSS Modules / Standard CSS
* **Table Management**: [@tanstack/react-table](https://tanstack.com/table/v8)

## Getting Started

Follow these steps to set up the project locally.

### Prerequisites

Make sure you have **Node.js** installed on your machine.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/vehicle-search.git](https://github.com/your-username/vehicle-search.git)
    cd vehicle-search
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

### Running the App

To start the development server:

```bash
npm run dev
